## 🎨 设计实现

### 配色方案
```css
--primary: #4CAF50;      /* 绿色主色 */
--secondary: #2196F3;    /* 蓝色辅色 */
--accent: #FF9800;       /* 橙色强调 */
--bg: #F5F5F5;           /* 背景灰 */
--card-bg: #FFFFFF;      /* 卡片白 */
--text-main: #333333;    /* 主文字 */
--text-sub: #666666;     /* 副文字 */
```

### 响应式断点
- 手机：0 - 640px（默认样式，单列布局）
- 平板：641px - 1024px（双列卡片）
- 桌面：1025px+（三列卡片 + 横向导航）
